Version 0.0.2-alpha released

This version doesn’t bring massive changes, therefore it is still a minor version change.

We’ve added the following:

  • Added a queue system for bruteforce commands. Premium users have priority in the queue.
  • bruteforce option to md5 command.
  • language option to bruteforce option of caesar and md5 commands.

The next version will come with new encoding/encryption algorithms so stay tuned!
