Ciphers

Some commands for the well known ciphers running around.

  • caesar - Manipulate a string with the Caesar cipher.
    • encode - Encode a string with the Caesar cipher.
    • decode - Decode a string with the Caesar cipher.
    • bruteforce - Bruteforce a string with the Caesar cipher.

General

Some of the general commands every bot has.

  • ping - Check if the bot is alive.
  • help - Get all the commands of the bot.

Hashing

Some basic hashing algorithms.

  • md5 - Manipulate a string with the MD5 hashing algorithm.
    • bruteforce - Bruteforce an MD5 hash.
    • hash - Hash a string with MD5.
